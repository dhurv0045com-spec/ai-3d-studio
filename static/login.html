What is AUREX

Language becomes
geometry.

AUREX is a text-to-3D generation engine that bridges natural language and physical form. Type anything — a dragon, a castle, a spaceship — and the system interprets your words through multiple AI pipelines to produce a real, downloadable 3D model.

Unlike image generators, AUREX produces actual geometry — mesh, materials, topology — renderable in any 3D engine, inspectable from every angle, editable in Blender.

How it works

Five stages.
One object.

AUREX cascades your prompt through a hierarchy of AI pipelines, each producing higher fidelity output.

01 / PARSE
Gemini interprets
AI reads your prompt and extracts object, style, material, color, and geometry complexity.
02 / SCRIPT
Code is written
Gemini generates a Blender Python script using only primitives — cubes, spheres, cones — arranged precisely.
03 / RENDER
Blender executes
Blender 4.2 runs headless, executes the script, exports a production-ready GLB file.
04 / PERSIST
Cloud storage
Every model is uploaded to Cloudinary. Persists across sessions, devices, and redeployments.
05 / VIEW
Three.js renders
Your object appears in the browser. Rotate, zoom, wireframe, edit, download. Full control.
