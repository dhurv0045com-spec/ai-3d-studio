What is AUREX

Language becomes
geometry.

AUREX is a text-to-3D generation engine. Type anything — a dragon, a spaceship, a medieval castle — and the system interprets your words through multiple AI pipelines to produce a real, downloadable 3D model.

Unlike image generators, AUREX produces actual mesh geometry — renderable in any 3D engine, inspectable from every angle, editable in Blender.

How it works

Five stages.
One object.

AUREX cascades your prompt through AI pipelines until the best result emerges.

01 / PARSE
Gemini interprets
AI extracts object, style, material, color, and geometry complexity from your words.
02 / SCRIPT
Code is written
Gemini writes a Blender Python script using only primitives, arranged precisely for your object.
03 / RENDER
Blender executes
Blender 4.2 runs headless, executes the script, exports a production-ready GLB file.
04 / PERSIST
Cloud storage
Every model uploads to Cloudinary. Persists across sessions, devices, and redeployments.
05 / VIEW
Three.js renders
Your object appears in the browser. Rotate, zoom, wireframe, edit, download.
